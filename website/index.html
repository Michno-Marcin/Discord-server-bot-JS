Strona główna Add on Discord

Bot functions

  • League of Legends player MMR check and profile search
  • Create polls and votes
  • Playing music from YouTube
  • Message cleanup
  • Kicking and banning users
  • Security of administrator commands through access control
The full list of commands and the arguments they take can be checked with !help

How to install the bot?

To add a bot, you must have a Discord account and administrator rights on the server where you want to install it. You can install the bot with default permissions or with administrator privileges (if you want the bot to run on all non-public channels).
